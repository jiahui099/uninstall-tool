# 一键卸载工具

把桌面上的垃圾软件图标拖进去，就能卸载它。妈妈也能用。

## 使用方法

### 方式一：直接下载 exe（推荐）
拿到我打包好的 `.exe` 文件，双击运行即可，不用装 Python。

### 方式二：如果拿到的是源码包
1. 确保电脑装了 Python 3.8+（Win10/11 自带或者去 python.org 下载）
2. 双击运行目录里的 `build.bat`
3. 脚本会自动装 PyInstaller 并打包
4. 打包完成后 `dist\一键卸载工具.exe` 就是最终的 exe

## 使用步骤

1. 打开程序
2. 把桌面上的垃圾软件快捷方式（或 .exe 文件）**拖到窗口里**
3. 程序会自动搜索卸载程序
4. 点 **"开始卸载"**，静默卸载

## 原理

- 拖入 `.lnk` 快捷方式 → 解析出目标路径 → 从注册表找卸载命令
- 拖入 `.exe` 文件 → 直接在注册表和同目录搜索卸载程序
- 执行 `msiexec /x` 或 `xxx.exe /S` 静默卸载

## 无法卸载的情况

- 程序没有标准卸载入口（绿色软件直接删目录即可）
- 需要管理员权限 → **右键 exe → 以管理员身份运行**
- 某些顽固软件（360/腾讯安全类）会拦截卸载，自己上官网找卸载工具

## 关于杀毒软件误报

用 PyInstaller 打包的工具会被某些杀毒软件标记为"未知程序"，这是正常现象。如果妈妈电脑上的安全管家报毒，告诉她这是我自己写的，不是病毒。
